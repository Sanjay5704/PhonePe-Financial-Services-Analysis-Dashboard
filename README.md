# 📊 PhonePe Financial Services Analysis Dashboard

A **multi-page Power BI dashboard** built to analyze digital payment behavior and financial service usage patterns across different PhonePe-style service categories.

This project focuses on transforming raw transaction data into a **clean, insight-driven analytics dashboard** covering key business areas such as **Money Transfer, Insurance, Loans, and Recharge & Bill Payments**.

---

## 🚀 Project Summary

This dashboard was created to simulate how a fintech company can monitor and analyze customer transaction behavior across multiple service offerings.

Instead of building just a single report page, this project was designed as a **business-facing analytics solution** with multiple dashboard views for different financial services.

### Core Focus Areas:
- Transaction trends over time
- User activity and service usage
- Financial service adoption
- Payment category analysis
- Failure reason tracking
- Data storytelling through interactive visuals

---

## 🎯 Business Objective

The goal of this project is to help stakeholders answer questions such as:

- Which financial services are most actively used?
- How are transaction patterns changing over time?
- Which service categories drive the most activity?
- Are users consistently engaging with recurring payment services?
- How can digital payment trends be visualized in a decision-friendly way?

---

## 🛠️ Tools & Technologies Used

- **Power BI** – Dashboard development and reporting
- **Power Query** – Data cleaning and transformation
- **DAX** – KPI calculations and business metrics
- **CSV / Structured Transaction Data** – Data source for dashboarding

# 📊 Dashboard Pages

This dashboard is divided into **5 dedicated pages**, each designed to analyze a different business function.

---

## 🏠 1. Home Dashboard

The **Home Page** provides a high-level overview of the platform’s digital payment performance.

### Key Elements:
- Total transaction KPIs
- User trend overview
- Service-wise contribution
- Failure reason analysis
- Overall transaction movement over time

### Why This Page Matters:
This page acts as the **executive summary dashboard**, allowing a quick understanding of overall platform activity before diving deeper into specific services.

![Home Dashboard](images/home.png)

---

## 🛡️ 2. Insurance Dashboard

The **Insurance Page** focuses on understanding how users engage with insurance-related services within the platform.

### Key Elements:
- Insurance transaction trend over time
- Service category comparison
- Transaction activity distribution
- Performance breakdown by relevant segments

### Why This Page Matters:
This view helps analyze the **adoption of financial protection products** within a digital payments ecosystem.

![Insurance Dashboard](images/insurance.png)

---

## 💳 3. Loan Dashboard

The **Loan Page** focuses on digital financial lending activity and related usage patterns.

### Key Elements:
- Loan transaction / disbursement trends
- Service contribution analysis
- Category-level breakdown
- Activity movement over time

### Why This Page Matters:
This dashboard helps highlight how users are interacting with **credit and lending services** on a digital platform.

![Loan Dashboard](images/loan.png)

---

## 💸 4. Money Transfer Dashboard

The **Money Transfer Page** represents one of the most important transaction categories in the platform.

### Key Elements:
- Money transfer transaction trend

- Core transaction contribution
- Service-level transaction movement

### Why This Page Matters:
This page focuses on the **core digital payment behavior** that drives most platform engagement.

## 📱 5. Recharge & Bills Dashboard

The **Recharge & Bills Page** analyzes recurring utility and recharge-based transactions.

This page highlights **consistent user behavior** and helps understand routine payment usage patterns.

# 📈 Key Insights Derived

Some of the major observations from the dashboard include:

- **Money Transfer** appears to be one of the most active and important service categories
- **Digital payment activity** shows strong engagement across multiple financial service areas
- **Recharge & Bill Payments** indicate recurring and stable usage behavior
- **Insurance and Loan services** suggest expanding adoption beyond core payments
- Service segmentation helps uncover different behavioral patterns across the platform
